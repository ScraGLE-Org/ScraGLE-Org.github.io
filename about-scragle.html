So... what is ScraGLE?

ScraGLE is a Scratch 3 based game development toolkit containing

  • a fully documented set of code demos
  • a library of game-ready assets (animated sprites, backgrounds, sounds & soundtracks)
  • curriculum materials
  • a fully playable example game that showcases the use of the code demos

Its aim is to bridge the gap that currently exists between starter projects aimed at beginners, such as those written by Code Club or Coder Dojo, and more professional game development using 3D game engines such as Blender, Kodu and Unity.

By developing ScraGLE we hope to provide kids with more knowledge and experience of game design and development challenges and techniques within the familiar Scratch 3 blocks-based environment. The curriculum material that we are developing alongside ScraGLE makes it ideal for use in code clubs, coder dojos, raspberry jams, in the classroom and at home.
